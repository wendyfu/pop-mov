# Popular Movies App
### for Udacity Android Developer Nanodegree Program

### Prerequisite
Edit this line in `app/build.gradle`:

`buildConfigField("String", "TMDB_API_KEY", "\"YOUR_API_KEY\"")`

Change `YOUR_API_KEY` with your own TMDB API KEY
